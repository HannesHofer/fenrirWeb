Fenrir

Welcome to Fenrir

Fenrir provides a user friendly way to route all traffic from configured trough a VPN tunnel. This is done via ARP Spoofing. Determined default GW on inputinterface is spoofed to configured device. Configuration is stored in /var/cache/fenrir/fenrir.db.

VPN Configuration

Configure various VPN tunnel in a user friendly overview. Set default connection tunnel or tunnel per device.

Dynamically adjustable routing

Dynamically set routing via VPN Tunnel. Configure Tunnel establishment (on-demand) or always connected. Enabled VPN routing with simple click. All devices in the same network range show up and can be routed trough VPN via simple click.
